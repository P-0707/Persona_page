Data Science Career Trends Analysis

Description!

Data Science Career Trends Analysis is an in-depth analytical project focused on exploring employment trends within the data science industry. By using a publicly available dataset from Kaggle, this project examines the key factors influencing data science careers, including salary, experience level, company size, and remote work policies.

The project was carried out using Python for data processing and Power BI for visualization. Through data cleaning, enrichment, and transformation, the analysis provides standardized salary data in USD, insights into remote work flexibility, and a comparison of company sizes based on compensation structures.

The dashboard presents a variety of visualizations to highlight insights, such as: - **Average Salary by Experience Level and Company Size**: Visualizes how compensation increases with experience and company scale. - **Remote Work Patterns**: Shows remote work ratios and trends across different experience levels, indicating a preference for remote work in small organizations and senior roles. - **Geographical Salary Distribution**: A heatmap displaying average salaries by employee residence and company location, showcasing regional disparities.

Designed to assist both employers and professionals in understanding employment trends, this project offers recommendations for salary structuring, remote work policies, and career planning. The analysis emphasizes the dynamic nature of data science careers, with insights valuable to those navigating the evolving job market.

This project is available for exploration on GitHub, where you can view the source code and interactive dashboard, providing a comprehensive look at the analysis behind global data science employment trends.
